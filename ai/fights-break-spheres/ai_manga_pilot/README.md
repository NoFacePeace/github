# 《斗之力，三段》AI 漫剧试播集图文分镜包

本目录是基于本地小说文本制作的个人试作分镜包，不包含成片、图片、配音或可公开商用授权文件。首集目标是验证短视频 AI 漫剧的叙事节奏、人物一致性和画面提示词可用性。

## Package Contents

- `text_cleaning_report.md`：源文本检查、试播集取材范围、清洗规则和已知异常。
- `episode_01_script.md`：90 秒试播集改编剧本，含旁白、对白、节奏和结尾钩子。
- `storyboard_episode_01.md`：22 镜头图文分镜表，字段按计划固定。
- `characters.md`：主要角色视觉设定、表情状态、提示词和一致性标签。
- `scenes_props_prompts.md`：场景、道具、特效与全局风格提示词。
- `acceptance_checklist.md`：剧情顺序、镜头数量、时长、字段完整性和版权边界检查。

## Creative Direction

- 形式：9:16 竖屏图文分镜包。
- 时长：约 90 秒。
- 镜头数：22 个。
- 风格：少年漫画风，强表情、强明暗、清晰线条、夸张斗气特效。
- 改编策略：保留人物关系与核心冲突，台词短视频化，不大段复刻原文。
- 试播集钩子：从“天才跌落”推进到“云岚宗来客”，结尾悬念停在纳兰嫣然真实来意。

## Production Notes

1. 先用 `characters.md` 固定人物外观和一致性标签。
2. 再按 `storyboard_episode_01.md` 逐镜生成画面。
3. 每个镜头使用中文提示词校准创意，再用 English prompt 迁移到图像模型。
4. 生成图片后，优先检查人物服装、年龄感、构图安全区和画面中文字乱码。
5. 后续若要公开视频或商业化，需要确认改编授权；没有授权时应改为原创化角色、势力与设定。
