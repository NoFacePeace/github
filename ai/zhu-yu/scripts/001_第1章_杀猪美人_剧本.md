# 第1章《杀猪美人》漫剧剧本

章节来源：`chapters/001_第1章 杀猪美人.txt`

改编形式：古言 AI 漫剧 / 第 1 集

建议时长：5-7 分钟

本集核心：用“雪天杀猪”的强画面立住樊长玉的人设，再用宋家退婚、孤女守家、救回神秘男子三条信息完成第一集钩子。

## 主要人物

- 樊长玉：屠户之女，父母新丧，被退婚，独自撑家，外柔内韧，手上有刀，心里有光。
- 樊长宁：樊长玉幼妹，五岁左右，病弱乖巧，依赖姐姐。
- 宋砚：樊长玉退婚竹马，新中举人，清雅疏离。
- 宋母：宋砚之母，嫌贫爱富，趋利避害。
- 赵大娘：邻家大娘，照看长宁，热心护短。
- 受伤男子：樊长玉从雪地里救回的年轻男人，重伤昏睡，身份成谜。

## 剧本正文

### 场 1：陈家院子 / 日 / 雪

细雪纷飞。

院中一口大锅沸着热水，白汽冲散落雪。地上泥水混着雪，门板架成临时案台，半扇猪肉铺在上面。

樊长玉站在案前，穿半旧素净袄裙，乌发挽起。她抬手，黑铁砍骨刀落下。

一声闷响。

猪骨断开，案板震颤。

屋里火塘边，陈家宾客围坐烤火，听见声音，齐齐往院中看。

宾客甲压低声音。

宾客甲：樊二家才办完白事，陈家怎还请长玉这丫头来杀年猪？

宾客乙：陈家同樊二家交情好。再说了，这丫头如今不出来做活，姐妹俩吃什么？

樊长玉似没听见，继续分肉。她动作干净利落，刀刃在雪光里一闪一闪。

宾客丙往院里瞥。

宾客丙：爹娘没了，大伯又惦记她家房地，宋家那婚事也黄了。一个姑娘家，难啊。

宾客丁声音更低。

宾客丁：我听说宋家退亲，是嫌她命硬。克死爹娘，还克胞妹。

火塘边一阵唏嘘。

樊长玉握刀的手停了一瞬。

她垂着眼，脸上没有表情，只把刀重新抬起。

砰。

又一块骨头利落断开。

旁白：腊月里，临安镇人人忙着过年。樊长玉忙着活下去。

### 场 2：旧事蒙太奇 / 日夜交错

一组快速闪回，穿插在樊长玉分肉的动作中。

街头雨雪，宋母带着年少宋砚跪在路边，求一口薄棺。

樊父樊母停下脚步，将钱袋递过去。

宋母含泪叩谢，拉着宋砚向樊家许亲。

画面一转，少年宋砚伏案读书，樊家送来肉食、点心、束脩银钱。

再转，宋砚高中举人，宋家门前车马热闹。

宋母换上新衣，神情从感激变成挑剔。

最后，宋母站在樊家门前，手里捏着八字帖，冷淡开口。

宋母：算命的说了，这八字合不得。真成亲，砚哥儿和我都要被她克着。

门内，樊长玉沉默站着。

画面切回陈家院子。

樊长玉收刀，吐出一口白气。

樊长玉低声：糟心事，莫想也罢。

### 场 3：陈家院子 / 日

猪肉分割完毕。

主人家把工钱递给樊长玉，又让人提来一桶猪下水。

主人：长玉，今日辛苦了。进屋吃口热的再走？

樊长玉接过工钱和木桶，摇头。

樊长玉：不了。家里刚办过丧事，我就不进屋讨嫌了。陈叔照拂生意，我记着。

主人想说什么，终究只点点头。

樊长玉背起装刀的皮褡裢，一手提木桶，一手攥着钱，转身走进风雪。

### 场 4：药铺与雪路 / 日 / 蒙太奇

药铺柜台上，药包一包包扎紧。

药铺伙计：两副药，一副给小的，一副给伤患。拿好。

樊长玉付钱，指尖在铜板上停了停，还是推了过去。

旁白：家里有个病弱的妹妹。隔壁还有个半只脚踏进鬼门关的男人。

雪野闪回。

昨日黄昏，樊长玉背着刀具从乡下归来。雪地里，一道血痕蜿蜒到路边。

她停步，看见一个浑身是血的男人倒在雪中。

男人脸上满是干涸血污，看不清模样，只剩微弱呼吸。

樊长玉蹲下，探了探鼻息。

她皱眉。

樊长玉：还活着。

风雪压下来。她咬牙，把男人背起。

画面切回药铺门口。

樊长玉把药包收进怀里，提起猪下水，继续往家走。

### 场 5：城西民巷 / 傍晚 / 雪后

城西民巷阴暗潮湿，墙根青苔斑驳，旧门窗透着霉气。

樊长玉刚走进巷口，迎面遇上宋母和宋砚。

宋母穿新裁冬衣，耳上金饰晃眼。宋砚一身鸦青长衫，清雅干净，与狭窄破旧的巷子格格不入。

樊长玉身上却还沾着血沫，旧袄打着补丁，背着刀褡裢，一手药包，一手木桶。

三人相对。

无人先开口。

宋母看见木桶里的血水，嫌恶地避开半步，用帕子掩了掩鼻。

樊长玉目光从宋母金戒指上扫过，又看向宋砚的新衣。

她没有停。

樊长玉：看路咧。

巷子太窄。擦身而过时，木桶边沿蹭过宋砚衣摆，血水在杭绸上拖出一大片暗红。

宋母脸色骤变。

宋母：你这不长眼的丫头！这可是杭绸！

樊长玉脚步不停，背影挺直。

宋砚低头看了眼衣摆，眼底情绪淡得看不清。

宋砚：母亲，算了。

宋母盯着樊长玉远去的背影，压着火气。

宋母：罢了。再过几日，咱们也搬离这穷酸地方了。

宋砚没有接话。

风从巷中穿过，吹起他衣摆上的血色湿痕。

### 场 6：樊家门前 / 傍晚

樊长玉刚到家门口，一个小小的身影从邻家跑出来。

长宁：阿姐！

长宁粉雕玉琢，笑起来缺一颗牙。她张开手就要扑过去。

樊长玉眼疾手快，提住她后领。

樊长玉：别碰，我身上脏。

长宁乖乖停住，仰头看姐姐，伸手接过药包。

长宁：我帮阿姐拿。

赵大娘从邻家出来，笑着迎上来。

赵大娘：长玉回来了。

樊长玉从桶里捡出一串猪肝，递给赵大娘。

樊长玉：大叔好这一口，您炒了给他下酒。

赵大娘接过猪肝，也不跟她客气。

赵大娘：正好。对了，你昨夜背回来的那个年轻人，醒过一回了。

樊长玉动作一顿。

樊长玉：醒了？那我一会儿去看看。

长宁眼睛亮亮的。

长宁：阿姐，那个大哥哥可漂亮了！

樊长玉失笑，摸摸她的小揪揪。

樊长玉：哪有拿漂亮说男子的？

长宁认真。

长宁：就是漂亮。

樊长玉望向隔壁紧闭的屋门，神情多了几分好奇。

### 场 7：樊家屋内 / 傍晚

樊长玉回屋，放下猪下水和药包。

屋里简陋，却收拾得干净。墙角挂着旧围裙，桌上放着缺口茶碗。

她换下沾血的旧袄，洗净手上血迹。水盆里的清水慢慢泛红。

她抬头，看见墙边还放着父母生前用过的旧物。

短暂停顿。

樊长玉没有让自己陷进去，拧干布巾，转身出门。

### 场 8：赵家偏屋 / 夜

冬日天黑得早。

偏屋门窗封得严，屋里混着草药味、血腥味、汗味和炭火气。

樊长玉推门进去，眉头轻轻皱了一下。

床上隆起一道人影。

屋内昏暗，只能看见那人轮廓。

樊长玉摸到桌边，点燃油灯。

一豆暖光亮起。

她端着灯回身。

光落在床上男人的脸上。

血污已被擦净。那人脸色苍白，眉眼清俊，长睫在眼下投出淡影。即便昏睡，唇也紧紧抿着，像是骨子里不肯服软。

樊长玉怔住。

她想起长宁方才的话。

长宁声音回响：那个大哥哥可漂亮了。

樊长玉低声喃喃。

樊长玉：还真是……

床上的男人睫毛微颤。

樊长玉屏住呼吸，灯火在她眼中轻晃。

画面定格在男人缓缓将醒的侧脸上。

字幕：她救回来的，究竟是什么人？

本集结束。

## 改编备注

- 第一章大量背景信息已改为火塘议论、旧事蒙太奇和街巷对照，避免长段旁白。
- 樊长玉的人设重点放在“刀快、心稳、护家、有分寸”四点上。
- 宋家母子的戏份保留克制感，用避让、掩鼻、血水弄脏新衣来呈现阶层羞辱。
- 结尾停在受伤男子露面，保留下一集“醒来对话”和身份悬念。
